enum barelog_policy_t

Enum of all the policies available.

Enumerator
SKIP 

When buffer full, ignore new events.

REPLACE 

When buffer full, replace with new events.

FLUSH 

When buffer full, flush it to shared memory.

DESTROY 

Destroy buffer when full.
